HTML FORMS

How do websites get information? You are probably familiar with log-in or sign-up forms, and how we all hate them in some way. This uses form, allowing web applications to interact with users.

Basic form of Form

Form has some attributes that allow elements to be controlled and handled. Below are examples:

  • action - sends the data to a page; can only act on one HTML page
  • method - specifies how data is sent through two ways:
    • GET - non-sensitive data sent to URL
    • POST - sensitive or large data sent to server
  • autocomplete - controls browser suggestions to autofill inputs (on or off)
  • name - provides a name to the form, similar to class and id
  • disabled -
  • novalidate - a bool attribute that disables automatic form validation when submitting
  • target - specifices where response from server should be displayed
